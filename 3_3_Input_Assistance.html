Input Assistance


This means that if you're filling out an important form (like for a purchase or legal document), the website should help you avoid mistakes. It should let you check your answers, correct errors, or undo the submission.
Check for:
  • Important forms: Are there any forms for legal commitments, financial transactions, or submitting test responses?
  • Error prevention: Does the form allow you to review, confirm, and correct information before submitting?
  • Reversible submissions: Can you undo the submission if you make a mistake?

This means that forms and other input fields should have clear labels or instructions so you know what to type or select.
Check for:
  • Form fields: Do all form fields have labels?
  • Instructions: Are there clear instructions for completing the form or inputting information?
  • Purpose: Is it clear what kind of information should be entered in each field (e.g., numbers, dates, text)?

This means that if the website finds a mistake in your input, it should suggest ways to fix it.
Check for:
  • Error detection: Does the website automatically detect errors (e.g., invalid email format, incorrect date)?
  • Suggestions: Are suggestions provided for correcting the error?

  • Sensitive forms: Are there any forms that handle legal, financial, or important data?
  • Error prevention: Are there mechanisms to prevent errors, such as input validation, confirmation steps, or the ability to undo submissions?
