Time-based Media


For prerecorded audio-only and prerecorded video-only media, the following are true, except when the audio or video is a media alternative for text and is clearly labeled as such:
Check for:
  • Prerecorded audio: An alternative for time-based media is provided that presents equivalent information for prerecorded audio-only content.
  • Prerecorded video: Either an alternative for time-based media or an audio track is provided that presents equivalent information for prerecorded video-only content.

Captions are provided and include all spoken words in a video and important sound effects. Captions can be closed, open, or a combination of the two.
Check for:
  • Captions: Are there accurate captions synchronized with the audio?

Audio descriptions explain important visual details that aren't in the audio.
Check for:
  • Audio descriptions: Are there audio descriptions for important visual information?
  • Media alternative (TBMA): If audio descriptions aren't possible, is there a text or audio version that provides the same information?

Provide real-time captions for live video content.
Check for:
  • Live captions: Are there accurate, real-time captions synchronized with the live audio?

Use extended audio descriptions if there's not enough time in the regular audio to describe all the important visuals.
Check for:
  • Extended audio descriptions: If there are long pauses in the audio, are extended audio descriptions used to explain visuals?

Provide real-time sign language interpretation for live video content.
Check for:
  • Media Alternative: Is there a time-based media alternative available?
