This means that when you tab to something on the page, it shouldn't automatically open a new window, play a sound, or take you to a different page without you doing anything else.
Check for:
  • Tab through page: Use the Tab key to move through the page. Does anything unexpected happen when an element receives focus?
  • Pop-ups: Do any pop-up windows appear automatically?
  • Sounds: Do any sounds play automatically?
  • Page changes: Does the page redirect or refresh unexpectedly?

This means that when you change something on the page (like typing in a form field or selecting an option), it shouldn't automatically take you to a different page or do something unexpected, unless it warns you first.
Check for:
  • Form fields: When you type in a form field, does the page refresh or redirect unexpectedly?
  • Dropdown menus: When you select an option from a dropdown menu, does anything unexpected happen?
  • Buttons: Do buttons cause unexpected actions?
  • Warnings: If there are actions that cause a change of context, are users warned beforehand?

This means that menus and other navigation tools should appear in the same place and order on every page, so it's easy to find your way around.
Check for:
  • Navigation menus: Do menus and other navigation elements appear in the same place on every page?
  • Order of items: Are the items in the navigation menus always in the same order?

This means that buttons and other interactive elements should look and work the same way throughout the website.
Check for:
  • Buttons: Do buttons that do the same thing look the same across different pages?
  • Icons: Are icons used consistently to represent the same actions?
  • Labels: Are labels and instructions clear and consistent?
