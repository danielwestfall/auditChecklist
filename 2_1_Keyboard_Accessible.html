Keyboard Accessible


This means you should be able to use the website or app with just your keyboard, without needing a mouse.
Check for:
  • Navigation: Can you use the Tab key to move between links, buttons, and form fields?
  • Interaction: Can you use the Enter or Spacebar keys to activate links and buttons?
  • Forms: Can you fill out forms and submit them using only the keyboard?
  • Media players: Can you control media players (play, pause, volume) with the keyboard?
  • No mouse traps: Are there any places where you get stuck and can't navigate away with the keyboard?

This means that if you can tab to something with your keyboard, you should also be able to tab away from it.
Check for:
  • Tab through content: Tab through all interactive elements. Can you always tab away from them?
  • Pop-ups and modals: If a pop-up or modal window appears, can you tab out of it or close it with the keyboard?
  • Embedded content: Can you tab out of embedded content like videos or maps?

This means that if the website uses single-letter keyboard shortcuts, you should be able to turn them off, change them, or use them only when you're focused on a specific part of the page.
Check for:
  • Single-letter shortcuts: Are there any single-letter shortcuts?
  • Options: Can you turn off the shortcuts, remap them, or are they only active on focus?
